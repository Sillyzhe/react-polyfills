# `universal-polyfill-context`

> Context polyfill for React/Rax

## Usage

```js
import Engine from 'universal-polyfill-engine';
import { createContext } from 'universal-polyfill-context';

import React from 'react';
Engine.set(React);

import Rax from 'rax';
Engine.set(Rax);
```
