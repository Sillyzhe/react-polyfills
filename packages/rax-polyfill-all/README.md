# `rax-polyfill-all`

> Polyfills for Rax new api

## Support

`>= Rax@0.6.5`

## Usage

```js
import {
  // ref
  createRef,
  forwardRef,
  // context
  createContext,
  // hooks
  withHooks,
  useRef,
  useState,
  useContext,
  useReducer,
  useEffect,
  useLayoutEffect,
  useMemo,
  useCallback,
  useImperativeHandle
} from 'rax-polyfill-all';
```
